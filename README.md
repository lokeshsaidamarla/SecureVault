# 🔐 SecureVault - Enterprise Password Manager

SecureVault is a secure password management system built using Spring Boot and JSP.  
It helps users store credentials securely and perform security checks.

## 🚀 Features
- Secure Login System
- Password Encryption
- Password Storage
- Safe Link Checker
- Facial Verification
- User Profile Management
- Secure Logout System

## 🛠 Tech Stack
- Java
- Spring Boot
- JSP
- MySQL
- HTML / CSS
- JavaScript

## 📂 Project Structure
src/
 ├── controller
 ├── model
 ├── repository
 ├── service
 └── jsp pages

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ / VS Code
3. Run the Spring Boot application
4. Access at: http://localhost:8085/
